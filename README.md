**3DProceduralTerrain:**

This is a 3D procedural terrain generator. It contains plain and desert biomes. The generator uses 2D perlin noise and combines several octaves in order to get good looking terrain.

**Controls:**

Press the left shift key to enable and disable your mouse. Enabling the mouse also disables all camera movement. Press the escape key to exit the program.

**Download instructions:**

1: Download the repository.

2: Download dependencies that are prompted by Visual Studio (project must be run in Visual Studio).

3: Set the configuration to **Debug** and **x86**.

4: Run it and allow up to 45 seconds for it to load (you will see a white screen while it is loading).
